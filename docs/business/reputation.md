# 声誉系统 (Reputation)

## 评分因子

| 因子 | 影响 | 说明 |
|------|------|------|
| 完成交易数 | + | 成功协商并完成的交易 |
| 取消/失败 | - | 协商中途退出或失败 |
| 投诉率 | - | 接到投诉的次数 |
| 响应时间 | + | 对提案的响应速度 |
| 履约率 | + | 实际到场的比例 |

## 评分范围

- **0-30**: 新注册 / 低信誉
- **31-60**: 中等信誉
- **61-85**: 高信誉
- **86-100**: 顶级信誉

## 声誉影响

- 匹配评分中占 10%
- 高声誉 Agent 在匹配中获得优先推荐
